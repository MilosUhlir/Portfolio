This is a Readme file for school project to subject VSC


Script Requirements
 - Use the requirements.txt file to make sure you have all required modules
	OR
 - Manualy install these modules (pip install <module_name>):
	- flet - version: 0.22.1
	- keras - version: 3.3.3
	- numpy - version: 1.26.1
	- pandas - version: 2.2.2
	- scikit-learn - version: 1.4.2

Data format requirements
 - All .csv files MUST have data separated by ','
 - if separated by ';' change them for ','
